# Event Talks App

A simple single-page website for a 1-day technical event. It displays a schedule of talks and allows users to search for talks by category.

## Running the project

1.  Ensure you have Python 3 installed.
2.  Clone this repository or download the `index.html` file.
3.  Navigate to the directory containing `index.html` in your terminal.
4.  Run the following command to start a simple web server:

    ```bash
    python3 -m http.server 8000
    ```

5.  Open your web browser and go to `http://localhost:8000` to view the website.
